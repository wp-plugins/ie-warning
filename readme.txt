Shows nice and simple warning to your visitors with Internet Explorer, supports customization and settings

== Description ==

Once the plugin is activated, the plugin shows the blog visitors a splash message on the screen indicating a message that IE does not render web pages correctly and they may want to download Opera and Mozilla Firefox browsers. Of course, for the users who like IE 7, they may include a message to upgrade their browsers to IE 7. Or any other browser, you can change the message and customize other settings.

Now it have two display models: splash (old) and Line-at-top (new, default). You can customize css for your needs

== Installation ==

Download and Install the plugin as usual into your WordPress directory. Activate it. Go to the settings and customize it. You are done.
